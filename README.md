# 🐄 Dairy Farm Management System

A simple and efficient web application to help dairy farmers manage cattle records, breeding history, health records, and milk/feed data. The system also includes voice commands and multilingual support for easy use in rural areas.

---

## 🚀 Features
- Manage Cattle Information (Tag Number, Breed, Age, Gender etc.)
- Record Milk & Feed Data
- Maintain Health Records & Diseases
- Track Breeding & Expected Calving Dates
- Voice Control for Navigation
- Multilingual Support (Kannada, Hindi, Tamil, Telugu, Marathi)

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML, CSS
- Speech Recognition API

### Backend
- Node.js + Express.js
- MySQL Database
- Sequelize ORM
- JWT Authentication
